• GET /api/[node|way|area]/?{PARAMS}

    Get elements, like node, way or area.

  • Parameters (PARAMS)

    Name Required Description Default Value Example
    q False Query key, where put the fields of query. Receive a list of fields q=[id=1]
    format False Format of response. It can be: {geojson, wkt} ("wkt" is in test fase) geojson format=geojson

    q parameter

    Name Required Description Default Value Example
    id False The ID of a element q=[id=1]

  • 1º Example of Request

    TEST GET http://localhost:8888/api/node/

    TEST GET http://localhost:8888/api/node/?format=geojson
